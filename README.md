# uispec

## Usage

```javascript
const uispec = require('uispec');
const myUispec = uispec({
  srcDir: 'uispec/',
  serveDir: '/_uispec',
  destDir: 'dest/uispec/'
  themaDir: 'uispec-thema/'
});

```

## API

`serve()`
`generate()`
`clean()`
`watch()`


## Commands

`uispec generate`
`uispec watch`
`uispec clean`
